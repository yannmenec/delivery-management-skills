# Benchmarks

Before/after comparisons that demonstrate the effectiveness of agent-assisted delivery management versus manual processes.

## Structure

Each benchmark documents:
- **Before**: The manual process (steps, estimated time, common gaps)
- **After**: The agent-assisted process (steps, time, coverage)
- **Comparison**: What the agent catches that humans typically miss
