> Pre-populates retrospectives with sprint facts, data-driven prompts,

This agent is planned for a future release. It will aggregate sprint 
events (PR durations, blocker history, estimation accuracy, team load 
patterns) into a data-rich "Sprint Story" that makes retrospectives 
more productive.
